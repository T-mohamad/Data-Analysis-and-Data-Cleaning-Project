## ğŸ”„ Data Cleaning Steps
1. **Handling Missing Values**
   - Drop or impute missing data using mean/median/mode.
2. **Removing Duplicates**
   - Identify and remove duplicate rows.
3. **Data Type Conversion**
   - Ensure columns have correct data types (e.g., dates, integers).
4. **Outlier Detection**
   - Use statistical methods (IQR, Z-score) to detect and handle outliers.
5. **Standardization & Normalization**
   - Scale numerical features for consistency.
6. **String Cleaning**
   - Trim spaces, fix casing, remove special characters.
